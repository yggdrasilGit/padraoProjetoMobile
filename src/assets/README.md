# README da pasta `/assets`

## Assets (Arquivos Estáticos)

Pasta destinada a arquivos estáticos como imagens, fontes, ícones e animações.

### Estrutura recomendada

```
assets/
  images/
    logo.png
    background.jpg
  fonts/
    Roboto-Regular.ttf
  icons/
    home.png
```

### Responsabilidade

* Organizar recursos visuais do aplicativo.
* Facilitar o acesso a imagens e fontes pelo código.

### Boas práticas

* Separar por tipo: images, icons, fonts, animations.
* Utilizar nomes de arquivos claros e descritivos.
* Manter apenas arquivos usados pelo app.
